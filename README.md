# dfwroofingcontractors.com

Lead generation site for DFW roofing contractors.
Built and maintained via Claude + MCP tools. Hosted on GitHub Pages.

## Repo: mcpdemo-com/dfwroofingcontractors-com
## Live URL: https://dfwroofingcontractors.com

## Page Structure
- `index.html` — Homepage with lead capture form
- `services/index.html` — Services page
- `areas/index.html` — Service areas (Dallas, Fort Worth, Plano, Frisco, etc.)
- `about/index.html` — About page
- `estimate/index.html` — Free estimate form (primary CTA)
- `blog/` — SEO content
- `sitemap.xml`
- `404.html`

## Deployment
All files deployed via `mcpcio:gitmanager_write_file`.
Custom domain: dfwroofingcontractors.com
Lead forms: Formspree
